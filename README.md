## 프로젝트 실행 명령어  💻
```
1️⃣ 레포지토리 클론
git clone https://github.com/1mptera/web-03-project01-jennajeh.git

2️⃣ 디렉터리 이동
cd web-03-project01-jennajeh

3️⃣ 소스 코드 컴파일
./gradlew compileJava
java -cp build/classes/java/main Fries

4️⃣ 프로그램 실행
./gradlew run

5️⃣ 테스트 코드 실행
./gradlew test
```
